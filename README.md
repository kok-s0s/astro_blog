AI 可以帮你生成代码，但它没有经历过你踩的那个坑、做的那个决策、绕的那段弯路。

这个博客存在的意义，就是把这些经历写下来。不是为了教谁，而是为了让自己真正想清楚——写作是检验理解深度最诚实的方式。

同时也留一个痕迹：某年某月，我在做这件事，我是这样思考的。

---

## 在写什么

- C/C++ 在实际工程里的实践与踩坑
- 制造业软件开发的日常（桌面 GUI、工业通信、跨平台）
- 读书笔记与技术摘要
- 偶尔的工具折腾记录

## 部署

站点由 GitHub Actions 构建并发布到 GitHub Pages：

- push 到 `main` 时自动发布
- 每天北京时间 10:00 定时重新发布
- 也可以在 Actions 页面手动触发 `Deploy GitHub Pages`
